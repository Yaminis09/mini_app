# mini_app

This is a simple full stack To-Do application where user can add, update, and delete tasks.
The main goal of this project was to understand backend architecture and connect it with a working frontend.



# Features
- Add new task
- Mark task as completed / undo
- Delete task
- View all tasks
- Health check endpoint

# Tech
- Backend:
-- Python (Flask)
-- SQLAlchemy
-- SQLite
-- Pydantic
- Frontend:
-- HTML
-- CSS
-- JavaScript (Fetch API)


# Dependency 
pip install flask flask-cors flask-sqlalchemy pydantic
-Create virtual environment in backend folder.
-Install dependency
run Python app.py
-open index.html
